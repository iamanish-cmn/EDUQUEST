# 🎓 EquQuest – AI-Powered Learning Platform

EquQuest is a modern AI-powered learning platform that enhances the educational experience through interactive quizzes, personalized learning, and intelligent assistance. The platform enables users to test their knowledge, monitor their progress, and receive AI-generated guidance to improve their understanding. Designed with a clean and responsive interface, EquQuest provides an engaging environment for students and lifelong learners.

## ✨ Features

* 🤖 AI-powered assistance using Google Gemini
* 📚 Interactive quizzes across multiple categories
* 📈 Personalized progress tracking
* 🏆 Global leaderboard
* 🎖️ Certificate generation for achievements
* 🔖 Bookmark important questions
* 🔥 Daily learning streaks
* 👤 Secure user authentication
* 💎 Premium membership support
* 🌙 Dark and Light mode
* 📱 Fully responsive design
